Häufige Fragen & Probleme

Server erstellen

Wie erstelle ich einen Server?

Im Dashboard unter „Neuen Server erstellen“.

Warum kann ich keinen Server erstellen?

Du hast dein Paketlimit erreicht.

Welche Versionen werden unterstützt?

Alle Versionen von 1.8 bis 1.20+.

Wie ändere ich die Version?

Server stoppen → Version ändern → Starten.

Warum startet mein Server nicht?

Meistens wegen falscher Version oder Mods.

Performance

Warum laggt mein Server?

Zu wenig RAM oder zu viele Plugins.

Wie bekomme ich mehr RAM?

Paket upgraden (Basic, Advanced, Titan, Omega).

Warum ist der TPS niedrig?

Plugins/Mods verursachen hohe Last.

Wie optimiere ich Paper?

Paper‑YML anpassen (später im Backend möglich).

Warum ist die CPU-Auslastung hoch?

Viele Entities oder Redstone‑Maschinen.

Mods & Plugins

Wie installiere ich Mods?

Im Dashboard unter „Plugins & Mods“.

Warum funktionieren meine Mods nicht?

Falscher Server‑Typ (z. B. Fabric statt Forge).

Wie lösche ich Mods?

Im Mods‑Bereich auf „Entfernen“ klicken.

Warum crasht mein Server mit Mods?

Mod‑Konflikt oder falsche Version.

Wie viele Mods kann ich installieren?

Hängt vom RAM ab (Titan/Omega empfohlen).

Plugins

Warum lädt ein Plugin nicht?

Falsche Version oder fehlende Abhängigkeit.

Wie installiere ich Plugins?

Im Dashboard unter „Plugins & Mods“.

Wie lösche ich Plugins?

Plugin auswählen → Entfernen.

Warum sind meine Permissions kaputt?

Fehler in der permissions.yml.

Warum lädt LuckPerms nicht?

MySQL‑Fehler oder falsche Version.

Netzwerk & Verbindung

Warum kann ich nicht joinen?

Server offline oder falsche IP.

Warum ist mein Ping hoch?

Standort weit entfernt oder Internet langsam.

Warum fliege ich vom Server?

Timeouts durch schlechte Verbindung.

Warum sehen Spieler den Server nicht?

Serverliste deaktiviert.

Warum ist die IP so lang?

Subdomain‑System (später Custom‑Domains möglich).

Dashboard

Warum lädt das Dashboard nicht?

Browser‑Cache löschen.

Warum sehe ich keine Server?

Du hast noch keine erstellt.

Warum kann ich nichts anklicken?

JavaScript deaktiviert.

Warum ist mein Rang falsch?

Logout → Login.

Warum ist mein Avatar weg?

Bild konnte nicht geladen werden.

Login & Account

Warum kann ich mich nicht einloggen?

Falsches Passwort.

Wie ändere ich mein Passwort?

Im Profil‑Bereich (später verfügbar).

Warum bekomme ich keine E‑Mails?

Spam‑Ordner prüfen.

Wie ändere ich meinen Namen?

Im Profil‑Bereich.

Warum ist mein Account gesperrt?

Zu viele Login‑Versuche.

Hosting & Pakete

Wie upgrade ich mein Paket?

Auf der Pakete‑Seite.

Warum ist mein Server langsam?

Free‑Paket hat begrenzte Leistung.

Wie viele Server kann ich haben?

Hängt vom Paket ab.

Warum ist Titan/Omega so teuer?

Premium‑Hardware & unbegrenzte Server.

Wie kündige ich?

Im Dashboard (später verfügbar).

Fehler & Crashes

Was bedeutet „Out of Memory“?

Zu wenig RAM.

Was bedeutet „Failed to bind port“?

Port wird bereits genutzt.

Was bedeutet „Invalid mod file“?

Mod beschädigt oder falsch.

Was bedeutet „Unsupported Java version“?

Version passt nicht zum Server‑Typ.

Was bedeutet „Plugin not found“?

Plugin fehlt oder wurde gelöscht.
